# Admin-Management

Admin dashboard page and full backend because admin manages users, agents, properties, appointments, and messages.

## Structure
- `frontend/` contains the selected HTML files, `css/style.css`, `js/script.js`, and an `images/` folder.
- `backend/realestate-backend/` contains the Spring Boot files related to this branch.

## How to use this branch folder
1. Copy this folder content into the matching GitHub branch.
2. Keep the same folder names: `frontend`, `backend`, `css`, `js`, and `images`.
3. Add your real image files into `frontend/images/` because only code files were uploaded here.
4. Commit and push to the matching branch.

## Suggested Git commands
```bash
git checkout Admin-Management
git add .
git commit -m "Update Admin-Management files"
git push origin Admin-Management
```
